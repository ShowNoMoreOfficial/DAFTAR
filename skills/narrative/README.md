# Domain 2: Narrative Intelligence (Yantri)

Narrative skills power Yantri's ability to transform signals into compelling content.

## Sub-domains

- `/editorial` — Topic selection, framing, branching, contrarian angles
- `/voice` — Psychological skill, diplomatic craft, hook engineering
- `/research` — Fact dossier building, web research, data argumentation
- `/audience` — Audience calibration, demographic targeting, diversification

## Integration

Yantri's existing multi-agent pipeline (Ingestion > Vectorization > Research > Strategist > Craftsman) loads these skills at each phase instead of using hardcoded logic.
