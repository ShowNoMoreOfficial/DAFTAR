# Domain 4: Platform Intelligence (Relay + Yantri)

Platform skills optimize content for each distribution channel.

## Sub-domains

- `/youtube` — Titles, descriptions, tags, end screens, shorts, playlists
- `/x-twitter` — Tweets, threads, visual anchors, algorithm awareness
- `/meta` — Carousels, reels, stories, Facebook groups, ads
- `/linkedin` — Professional tone, articles vs posts, creator mode
- `/seo` — Keywords, on-page, content, technical, video SEO
- `/ppc` — Google Ads, Meta Ads, campaign analytics, retargeting
