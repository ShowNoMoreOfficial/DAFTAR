## Instructions

Architect X/Twitter threads that deliver deep analysis in a binge-readable format. Threads are the platform's long-form content equivalent.

### Thread Structure

#### Opening Tweet (Most Critical)
- Must work as a standalone tweet — many people only see Tweet 1
- Include the hook + 🧵 emoji to signal thread
- State what the reader will learn: "5 points mein samjhiye" / "Here's what nobody's reporting"
- Example (Squirrels): "India just made a move that'll cost $4.2B. But here's what nobody's reporting: it was the right call. A thread on why 🧵"
- Example (Breaking Tube): "Pakistan ka new budget analysis — 5 points mein samjhiye kya-kya gadbad hai 🧵"

#### Body Tweets (The Substance)
- One idea per tweet — don't cram multiple points
- Number the points: "1/" or "1." for scanability
- Alternate between: data tweet → analysis tweet → implication tweet
- Include a visual every 3-4 tweets (chart, image, screenshot)
- Each tweet should be readable without context of previous tweet (people jump in mid-thread)

#### Closing Tweet
- Summarize the "so what" — the bottom line
- Include CTA: link to video, follow prompt, or question for replies
- Example (Squirrels): "Bottom line: three scenarios, each with different odds. None comfortable. Full analysis with charts in the video → [link]"
- Example (Breaking Tube): "Bottom line: Pakistan ka budget survival mode mein hai. India ke liye iska matlab — detailed analysis video mein. Link 👇"

### Thread Length Guidelines
| Content Type | Optimal Length | Maximum |
|---|---|---|
| Breaking news analysis | 5-7 tweets | 10 |
| Deep dive / explainer | 8-12 tweets | 15 |
| Data story | 6-8 tweets | 10 |
| Quick take | 3-5 tweets | 7 |

### Thread Formatting Rules
- Use line breaks within tweets for readability
- Bold key numbers/claims using CAPS for single words (not entire sentences)
- Use → for logical flow: "More imports → higher deficit → weaker rupee"
- Avoid hashtags in body tweets (clutters the analysis) — save for last tweet
- Thread numbering: "1/10" or just "1." — be consistent

### Visual Strategy for Threads
- Tweet 1: Compelling image or thumbnail-style graphic
- Every 3rd tweet: Chart, data visualization, or screenshot
- Final tweet: Video thumbnail with link
- Visuals increase thread engagement by 50%+ and break up text walls

### Cross-Promotion
- Pin the thread on profile if it's performing well
- Share thread link in YouTube community post
- Quote-tweet your own Thread Tweet 1 with a teaser 6-12 hours later (catches different timezone)
