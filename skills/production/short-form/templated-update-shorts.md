## Instructions

Create repeatable, templated short-form content for daily/weekly updates. Speed through templates, not custom production.

### Template Types

**"Today in 60 Seconds"**: Daily news roundup
- 3 stories, 20 seconds each
- Template: Hook → Story 1 → Story 2 → Story 3 → CTA
- Visual: Split-screen or swipe transitions
- Audio: Consistent music bed, anchor voiceover

**"Market Minute"**: Daily financial update
- Key indices, currency, major movers
- Template: Numbers on screen with brief commentary
- Always end with "what to watch tomorrow"

**"This Week Explained"**: Weekly roundup
- Top 5 stories of the week in 60 seconds
- Template: Ranked list with 10-second treatment each
- Visual: Countdown format with bold numbering

**"One Stat That Matters"**: Single data point deep dive
- One surprising statistic + 45-second context
- Template: Big number reveal → "Here's why" → Implication
- Visual: Animated number + supporting graphic

### Template Production Rules
- Design templates once, reuse daily — consistency builds habit
- Keep visual identity exactly the same every day — recognizable in the feed
- Anchor recording can be batch-recorded (5 days at once)
- Graphics are fill-in-the-blank — only data changes, not design
- Total production time per template: 30-45 minutes
