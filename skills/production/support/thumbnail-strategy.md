## Instructions

The thumbnail is 50% of whether someone clicks. Design 3 concepts, A/B test the best.

### Thumbnail Design Principles

**1. Readability at small size**: Most viewers see thumbnails on mobile at ~120px wide
- Maximum 5 words of text
- Font size: minimum 48pt in the design file
- High contrast: light text on dark background or vice versa

**2. Emotional trigger**: The thumbnail must create an emotion
- Curiosity: "What does this chart mean?"
- Concern: "This affects me"
- Surprise: "I didn't expect that number"

**3. Visual hierarchy**: Eye goes to: face → number → text → background
- Include a face (anchor or relevant person) when possible — 2x click-through
- Large, bold numbers for data-driven content
- Color-code: Red for urgency/loss, green for growth/positive, blue for analysis

### Thumbnail Formula by Content Type

**Explainer**: Face (expression of insight) + bold stat + 3-word context
- Example: Anchor looking at camera + "₹4.2 LAKH CRORE" + "India's Hidden Cost"

**Quick Take**: Face (concerned expression) + flag/icon + 2-word hook
- Example: Anchor pointing + Indian flag + "OIL CRISIS?"

**Data Story**: Chart/graph + highlighted number + brand colors
- Example: Rising line graph + "85% IMPORTS" + red highlight

### Brand-Specific Styles
- **The Squirrels**: Clean, data-forward, blue/white palette, analytical feel
- **Breaking Tube**: Bold, high-energy, red/yellow accents, urgency feel

### A/B Testing
- Always create 3 variants with different visual approaches
- YouTube allows thumbnail A/B testing — use it for every video
- Track: click-through rate (CTR) per thumbnail variant
- Feed winners back into learning log
