## Instructions

Prepare content for localization across languages and cultural contexts.

### Current Localization Needs
- **English → Hindi/Hinglish**: The Squirrels content adapted for Breaking Tube audience
- **Hindi → English subtitles**: Breaking Tube content made accessible to non-Hindi speakers
- **India → Global**: Indian-focused content adapted for diaspora/international viewers

### Localization Levels

**Level 1 — Subtitles**: Add translated subtitles to existing content
- Cheapest, fastest, minimum viable localization
- Auto-generated + human review for accuracy
- Always provide English subtitles for Hindi content

**Level 2 — Voiceover**: Replace audio with new language voiceover
- Medium effort, good for expanding audience
- Script must be translated, not just transcribed
- Cultural references need adaptation, not literal translation

**Level 3 — Full adaptation**: Create new content inspired by the original
- Highest effort, highest impact
- Same fact dossier, different narrative for different audience
- This is what The Squirrels → Breaking Tube adaptation looks like

### Translation Notes Format
- Flag idioms that don't translate literally
- Mark India-specific references that need context for international audience
- Note currency conversions (₹ → $ where relevant for diaspora)
- Identify cultural references that need explanation or substitution
