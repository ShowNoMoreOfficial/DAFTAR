## Instructions

Create detailed briefs for every motion graphic element in a video.

### Brief Template Per Element
```
GRAPHIC #1
Type: Bar chart (animated)
Data: India semiconductor imports by year (2020-2026)
Source: Ministry of Electronics data, verified in fact dossier
Placement: 01:15-01:35 (20 seconds on screen)
Animation: Bars grow from left to right, final bar (2026) highlighted in accent color
Text: "India's Chip Import Bill" as title, Y-axis in $ billions
Brand colors: Use #2E86AB for primary bars, #A23B72 for highlighted bar
Notes: Numbers should count up as bars grow. Final number hold for 3 seconds.
```

### Graphic Types & When to Use

| Type | Use When | Duration |
|------|----------|----------|
| Bar chart | Comparing quantities across categories/time | 10-20s |
| Line chart | Showing trends over time | 10-15s |
| Map overlay | Geographic data, trade routes, conflict zones | 15-25s |
| Counter/ticker | Single dramatic number reveal | 5-10s |
| Split screen | Before/after or two-way comparison | 10-15s |
| Timeline | Chronological events | 15-20s |
| Org chart/flowchart | Relationships, processes | 10-20s |
| Pie/donut chart | Proportions (use sparingly) | 8-12s |

### Design Rules
- All graphics must be readable on mobile (minimum font size 24pt in final output)
- Use brand color palette — never introduce new colors
- Animate data — static charts are boring; growth, movement, reveals
- Source attribution: Small text in corner of every data graphic
- Don't overload: maximum 5 data points per graphic

## Learning Log

### Entry: Initial
- Animated counter reveals ("India lost ₹4.2 lakh crore") are the highest-engagement graphic type
- Map animations hold attention for 15-20 seconds — excellent for geopolitical content
- More than 3 graphics per 5 minutes causes "chart fatigue" — space them out
