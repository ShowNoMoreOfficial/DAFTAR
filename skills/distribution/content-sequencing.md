## Instructions

Sequence content releases across platforms to maximize total reach, engagement, and audience flow between platforms.

### The Sequencing Principle

One story → multiple platforms → released in strategic order → each release drives traffic to the next.

### Standard Sequence Pattern

```
T-4h: Community post / Story teaser → "Coming tonight"
T-2h: X/Twitter teaser tweet → Hook + "Full analysis at 7 PM"
T=0:  YouTube long-form publish → Primary release
T+1h: X/Twitter thread → Key insights from the video
T+2h: YouTube Shorts (1-2) → Best clips extracted
T+4h: Instagram Reel → Adapted for vertical format
T+6h: Instagram Carousel → Data points from the video
T+12h: LinkedIn post → B2B angle extraction
T+24h: YouTube Shorts (additional) → Different angles
T+48h: X/Twitter quote-tweet own thread → Catches different timezone
```

### Sequence Variations by Content Type

#### Breaking News Sequence
```
T=0:  X/Twitter → First reaction (speed wins)
T+15m: Instagram Story → Quick take
T+2-4h: YouTube quick take → Full analysis
T+4h: X/Twitter thread → Detailed breakdown
T+5h: YouTube Shorts → Best moment clips
T+6h: Instagram Reel → Adapted clip
```

#### Explainer/Deep Dive Sequence
```
T-24h: X/Twitter poll → "What aspect should we cover?"
T-4h: Community post → Teaser with preview stat
T=0: YouTube long-form → Primary release
T+2h: X/Twitter thread → Extended analysis
T+6h: Instagram carousel → Data visualization
T+24h: LinkedIn article → Professional angle
T+48h: YouTube Shorts → Individual sections as standalone clips
```

#### Series/Multi-Part Sequence
```
Episode 1 → Full sequence above
Between episodes: Daily teasers on X/Twitter and Stories
Episode 2 → Full sequence, but promote Episode 1 as prerequisite
After series: Compilation Short + Thread summarizing all episodes
```

### Cross-Platform Flow Logic

#### YouTube → Other Platforms
- YouTube is the source — other platforms amplify and extract
- Every platform post should funnel back to YouTube (link in bio, reply, description)
- Don't give away the full analysis on other platforms — leave reason to click through

#### X/Twitter → YouTube
- Tease the insight, don't deliver it fully
- "The numbers are worse than you think. Full breakdown: [link]"
- Thread should cover 60% of the analysis — enough to hook, not enough to satisfy

#### Instagram → YouTube
- Visual hook (thumbnail/clip) + caption that drives curiosity
- "Full 12-minute analysis on our YouTube — link in bio"
- Reels should be exciting enough to follow the account, driving eventual YouTube discovery

### Sequencing Anti-Patterns
- **Simultaneous dump**: Publishing everywhere at once — platforms compete with each other
- **Reverse funnel**: Publishing full content on secondary platforms first — no reason to visit YouTube
- **Orphan content**: Publishing on one platform with no cross-promotion — leaves reach on the table
- **Over-sequencing**: Too many posts about the same topic — audience fatigue after 48 hours
