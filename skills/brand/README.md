# Domain 7: Brand & Business (Daftar Core + Finance)

Brand skills define identity, voice, and business rules for each client brand.

## Sub-domains

- `/identity/{brand-slug}/` — Per-brand identity, platforms, audience, voice examples
- `/identity/_template/` — Template for onboarding new brands
- `/business` — Brand health, competitive positioning, partnerships
- `/finance` — Invoicing, revenue tracking, budget management

## Adding a New Brand

1. Copy `/brand/identity/_template/` to `/brand/identity/[new-brand]/`
2. Fill in identity.md, platforms.md, audience.md
3. Create Client + Brand records in Daftar database
4. The entire skill ecosystem immediately works for the new brand
