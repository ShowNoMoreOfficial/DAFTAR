## Instructions

Understand and work with (not against) X/Twitter's algorithm to maximize organic reach. Algorithm changes frequently — this skill file should be updated when changes are detected.

### Current Algorithm Signals (Positive)

#### Content Signals
- **Replies received**: Highest weight signal — tweets that generate discussion get amplified
- **Bookmarks**: Strong signal of valuable content (private engagement)
- **Quote tweets**: Strong signal — indicates content worth adding to
- **Retweets**: Moderate signal — distributes to retweeter's followers
- **Likes**: Baseline positive signal
- **Time spent reading**: Longer tweets/threads that hold attention score higher
- **Link clicks**: If tweet includes a link, clicks indicate interest

#### Account Signals
- **Consistent posting**: Regular posting (3-8 tweets/day) maintains algorithmic standing
- **Profile completeness**: Verified accounts, complete bios get slight boost
- **Follower engagement rate**: High-engagement followers amplify reach more
- **Topic authority**: Consistently posting about the same topics builds topical authority

### Algorithm Penalties (Avoid These)

#### Content Penalties
- **External links in tweet text**: Links to YouTube, websites etc. get suppressed — put links in replies instead
- **Excessive hashtags**: More than 2-3 hashtags signals spam
- **Identical/repetitive content**: Posting same text multiple times gets suppressed
- **Engagement bait phrases**: "Like and retweet if..." explicit engagement requests get penalized

#### Behavioral Penalties
- **Posting bursts**: 10+ tweets in 30 minutes looks automated
- **Follow/unfollow patterns**: Aggressive following looks spammy
- **Mass tagging**: Tagging 5+ accounts in one tweet
- **Deleted tweets**: Frequently deleting tweets may signal low-quality content

### Posting Strategy Based on Algorithm

#### Optimal Posting Cadence
- **Breaking Tube**: 8-12 posts/day (mix of tweets, replies, quote tweets)
- **The Squirrels**: 5-8 posts/day (fewer but higher substance per tweet)
- Space tweets minimum 30-45 minutes apart
- Best times for Indian audience: 8-10 AM, 12-2 PM, 7-10 PM IST

#### Link Posting Workaround
Since external links get suppressed:
1. Post the tweet without the link
2. Immediately reply to your own tweet with the link
3. Or: Post the tweet, then edit to add the link after initial engagement

#### Thread Optimization
- Threads get rewarded when each tweet in the thread gets engagement
- Front-load the best content — later tweets get fewer impressions
- Include visuals in at least 30% of thread tweets

### Monitoring Algorithm Changes
- Track impressions-per-tweet weekly — sudden drops may indicate algorithm changes
- Follow platform update announcements
- A/B test posting strategies monthly
- Log any detected changes in the Learning Log
