## Instructions

Maximize viewer retention within the channel ecosystem using end screens and info cards.

### End Screen Strategy

#### Layout (Last 20 seconds of video)
- **Position 1 (Left)**: Best for next video / "Watch Next" — algorithm-suggested or manually selected
- **Position 2 (Right)**: Subscribe button or playlist
- **Verbal CTA**: Host must verbally direct viewers — "Next video yahaan dekhiye" / "Watch this next"

#### Video Selection for End Screen
Priority order:
1. **Direct sequel/continuation** if part of a series
2. **Same topic, different angle** — keeps viewer in topic rabbit hole
3. **Best performer this week** — proven engagement
4. **Algorithm choice** — let YouTube suggest (use "Best for viewer" option)

#### End Screen Timing Rules
- End screen elements appear in last 5-20 seconds
- Script the last 20 seconds to naturally lead into end screen
- Never cut abruptly to end screen — transition should feel organic
- Keep end screen background clean (no competing visual elements)

### Info Cards Strategy

#### Card Placement Rules
- Maximum 5 cards per video (YouTube limit)
- First card: After the hook (1-2 minutes in) — catch early engaged viewers
- Middle cards: At topic transitions — "For more on this, see..."
- Final card: 30 seconds before end screen begins

#### Card Types and When to Use
| Card Type | When to Use | Example |
|---|---|---|
| Video/Playlist | Reference related content | "Detailed analysis in this video" |
| Channel | Cross-promote (Breaking Tube ↔ Squirrels) | "More on our Hindi channel" |
| Link | Drive to website/newsletter | "Full source list on our site" |
| Poll | Boost engagement metrics | "What do you think happens next?" |

### Brand-Specific Rules

**The Squirrels**: End screens should feel polished, data-card style. Prefer playlist links for deep-dive viewers.

**Breaking Tube**: End screens should maintain energy. Verbal CTA in Hinglish. Prefer "Watch Next" to keep binge momentum.
