## Instructions

Optimize YouTube descriptions for discovery (SEO), engagement (clicks to links), and viewer experience (timestamps, context).

### Description Structure

#### Template — The Squirrels
```
[2-3 sentence summary — what the video covers and why it matters]

📊 Key data points covered:
• [Data point 1]
• [Data point 2]
• [Data point 3]

⏱️ Timestamps:
[00:00] Introduction
[MM:SS] [Section title]
...

📖 Sources:
[Numbered source list with links where available]

🔗 Follow The Squirrels:
[Social links]

#TheSquirrels #[TopicHashtags]
```

#### Template — Breaking Tube
```
[Hindi-English summary — 2 lines, conversational tone]

Timestamps:
[00:00] [Hindi/Hinglish section title]
...

#BreakingTube #[TopicHashtags] #IndianPolitics

Follow Breaking Tube:
[Social links]
```

### SEO Rules
- First 2-3 lines appear above the fold — front-load keywords and value proposition
- Include primary keyword naturally in first sentence
- Use 3-5 relevant hashtags (YouTube indexes these for search)
- Description length: 200-500 words optimal for SEO
- Include 2-3 related keyword phrases naturally in the body

### Timestamps
- Always include timestamps — improves viewer experience and SEO
- YouTube creates chapter markers from timestamps automatically
- First timestamp must be [00:00]
- Minimum 3 timestamps for videos over 5 minutes

### Link Strategy
- Pin most important link (newsletter, related video) above the fold
- Don't overload with links — 3-5 maximum
- Use UTM parameters for trackable links

### Hashtags
- Maximum 15 hashtags, but 3-5 targeted ones perform better
- First 3 hashtags appear above the video title on mobile
- Mix: 1 brand hashtag + 2-3 topic hashtags + 1 trending if relevant
