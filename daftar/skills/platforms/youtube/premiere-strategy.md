## Instructions

Use YouTube Premieres strategically for high-impact videos to create event-like viewing experiences and boost Day 1 metrics.

### When to Premiere (vs. Standard Publish)
| Criteria | Premiere | Standard Publish |
|---|---|---|
| Tier S/A content | ✅ | |
| Breaking news / time-sensitive | | ✅ (speed > event) |
| Series finale or special | ✅ | |
| Regular quick takes | | ✅ |
| Tier B/C content | | ✅ |

**Rule**: Premiere sparingly — max 1-2/month. Overuse dilutes the "event" feeling.

### Premiere Setup

#### Countdown Duration
- **2 hours**: Standard for planned content — enough time to notify audience
- **30 minutes**: For semi-breaking content that still benefits from event format
- **Scheduled (next day)**: For maximum promotion runway on major content

#### Pre-Premiere Promotion
1. **Community post**: Announce premiere 4-6 hours before
2. **X/Twitter**: Thread or tweet with premiere link 2-3 hours before
3. **Instagram Story**: Countdown sticker 1 hour before
4. **YouTube notification**: Ensure premiere is set to notify subscribers

### Live Chat Management
- Host or team member must be present in live chat during premiere
- Pin a welcome message with context about the video
- Respond to early comments — first 50 comments set the tone
- Moderate aggressively: remove spam, hate speech, off-topic promotion
- Use Super Chat moments if enabled (revenue opportunity)

### Brand-Specific Premiere Style

**The Squirrels**: Professional, measured. Pin a comment with source links. Engage with substantive questions in chat.

**Breaking Tube**: High energy. Bhupendra or team actively chatting. Hinglish engagement. Create excitement — "Abhi dekhiye, full analysis aa rahi hai!"

### Post-Premiere
- First 24 hours after premiere are critical for algorithm — the premiere concentrates initial views
- Share premiere replay link across platforms immediately after
- Engage with comments for first 2 hours post-premiere
