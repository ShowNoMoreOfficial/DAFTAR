## Instructions

Define and maintain edit templates for recurring content formats, reducing production time.

### Template Library

**Template: Daily News Short (45s)**
```
[0:00-0:02] Brand intro animation (5 frames)
[0:02-0:05] Text overlay: today's headline
[0:05-0:35] Anchor voiceover + B-roll/graphics (content zone)
[0:35-0:42] Key takeaway text card
[0:42-0:45] Brand outro + CTA text
```
Automation: 70% — voice + content zone needs filling, rest is template

**Template: Data Carousel (8 slides)**
```
Slide 1: Brand header + bold headline
Slide 2: "Here's what happened" + key event
Slides 3-6: One data point each (number + context + source)
Slide 7: "Why it matters" + implication
Slide 8: CTA + brand handle + hashtags
```
Automation: 80% — fill in data points and headline, template handles design

**Template: X Thread (10 tweets)**
```
Tweet 1: Hook + summary (max 280 chars, no link)
Tweets 2-3: Context (what happened)
Tweets 4-7: Analysis (one insight per tweet)
Tweet 8: Data point with image
Tweet 9: Takeaway
Tweet 10: CTA + link to full video
```
Automation: 60% — AI generates draft, human reviews for voice/accuracy

### Template Maintenance
- Review template performance monthly
- Update templates when platform algorithms change
- Version control: keep previous templates archived
- A/B test template variations quarterly
