# Domain 6: Analytics & Learning (HOCCR + GI)

Analytics skills power the self-improving learning loop.

## Sub-domains

- `/performance` — Attribution, benchmarking, A/B testing, audience evolution
- `/revenue` — Revenue attribution, forecasting, monetization strategy
- `/feedback` — Sentiment loops, skill scoring, learning cycle reports

## The Learning Loop

Content Published > Platform APIs return metrics > Performance attribution > Skill scoring > Learning Log updates > Next content cycle benefits from learnings
