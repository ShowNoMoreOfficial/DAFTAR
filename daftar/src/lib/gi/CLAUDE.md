# GI — General Intelligence Copilot Context

## What The GI Is
Persistent, role-aware, context-aware AI inside Daftar on every screen. Organizational nervous system.

## Graduated Autonomy (4 Tiers)
- Tier 1 (Inform): Observes and reports. No action.
- Tier 2 (Suggest): Proposes action with one-tap approve.
- Tier 3 (Act & Notify): Takes action, tells you. Offers undo.
- Tier 4 (Act Silently): Handles without notification.
Admin configures per action type via GITierAssignment table.

## Must Do
- Answer questions about tasks, deadlines, workload, deliverables from live DB
- Reassign tasks, extend deadlines, rebalance workload (when tier allows)
- Surface proactive insights on dashboard
- Respect role boundaries
- Know what screen/context the user is on

## Must Never Do
- Expose individual performance struggles to peers
- Create anxiety through excessive deadline reminders
- Make people feel replaceable
- Override human judgment on creative decisions without explicit config
